# 📝 Notes Saver CLI (Python)

A simple and clean command-line notes application built with Python.  
This project helps you quickly save, view, and delete notes directly from your terminal.

---

## 🚀 Features

- ➕ Add new notes  
- 📋 View all saved notes  
- ❌ Delete specific notes  
- 💾 Automatically saves notes in a text file

## ⚙️ How It Works

- Notes are stored in a simple notes.txt file  
- Each note is saved on a new line  
- You can manage everything from the terminal
